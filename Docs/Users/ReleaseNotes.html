WinMerge 2011.004.525 Release Notes

This is a stable release of WinMerge 2011.
Besides fixing bugs, it includes the following functional enhancements:

  • The option to automatically check out read-only files from Visual SourceSafe or Rational ClearCase now also exists for Team Foundation Server.
    When the checkout fails, you have a new option to continue as if no version control system was selected, i.e. try to overwrite the file or save it elsewhere.

  • File filters have been enhanced as requested here.

  • When choosing Tell from filename extension as the Compare As method, WinMerge selects the effective Compare As method based on the filename extension.
    Filename extensions are mapped to file transform monikers as per section FileTransforms of Supplement.ini, which you can edit to meet your particular requirements.

  • The option to include the windows shell context menu in the folder compare view's context menu now also exists for the textual file comparison view.
    This enhancement was contributed by magelu and is tracked here.
    Known issue: The binary file comparison view disregards this option. This may or may not change in future.

  • Folder compare report types now also include Semicolon-separated list to help cope with applications which prefer it that way.
