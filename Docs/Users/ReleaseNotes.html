WinMerge 2011.001.527 Release Notes

This is the first stable release of WinMerge 2011. It fixes a number of bugs and regressions that have been found in the last beta, and finally includes a compiled help file created from a revised version of the WinMerge 2 DocBook sources. If you have worked with WinMerge 2, and want to learn about the most relevant differences, section File transforms and the Frequently asked questions 3.6 and 7.2 are of particular interest.

WinMerge 2011 still lacks an equivalent for WinMerge 2's prediffer plugins. My vague plan with this is to provide line filters with knowledge of both side's content. This would kind of demand a bit more expressive power than regular expressions can offer, but JavaScript/VBScript are good options.

Jochen Neubeck, December 2011
