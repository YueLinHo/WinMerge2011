WinMerge 0.2011.001.497 Release Notes

This is intended to be the last beta before going stable.

A big win is xdoc2txt, which enables WinMerge to compare the textual content of a large variety of office-type documents. Thanks to cecilyen for suggesting it in WinMerge Open Discussion forum.

The MediaInfo file transform scriptlet was already present in the preceding release, but has been revised to use an environment variable to locate the MediaInfo.exe.

Steps to get xdoc2txt and MediaInfo operational:

  • Obtain a copy of d2txt*.zip, and extract it into a folder of your choice.
  • Start WinMerge, invoke Edit → Options from the menu, go to page System, and look for the Supplement folder. In this folder, look for a file named Supplement.ini. In section Environment, edit item xdoc2txt to reflect the folder where the xdoc2txt.exe resides.
  • Do likewise to tell WinMerge where to find the command line version of MediaInfo. Be sure to obtain the CLI version, not the GUI one.

Another new feature that has long been on my mind is archive vs. folder compare.

Jochen Neubeck, December 2011
