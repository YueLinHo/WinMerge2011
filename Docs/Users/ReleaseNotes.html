WinMerge 0.2011.003.157 Release Notes

This is a beta release of WinMerge 2011. Significant new features since latest stable release include:
  • Line filters exist in a new flavor referred to as early-applied line filters:

    WinMerge.chm → Using Filters → Using line filters → Early-applied line filter syntax

  • The Open dialog's Include Subfolders option can be adjusted to ignore folder structure:

    WinMerge.chm → Quick start → Selecting files and folders to compare → 5.

  • Editor scriptlets are back:

    WinMerge.chm → Plugin scriptlets

  • Folder compare supports customizable background colors to make differences more visible.
    This feature can be configured through Edit → Options → Colors → List.
  • The location bar remains responsive during drag-drop editing. This greatly eases drag-drop moving of text over long distances.
    While in drag-drop mode, you can stop scrolling by holding down the Shift key.
  • ClearCase .mrgman files can be opened to a folder compare window, by dropping them onto the WinMerge main window.
    The header bars provide context menus to switch between base, source, and destination versions of the referenced files.

Bill Evjen's Blog gives advice in case WinMerge.chm fails to display content.
