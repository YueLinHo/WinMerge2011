WinMerge 2011.004.185 Release Notes

This is a stable release of WinMerge 2011.
Besides fixing some bugs found in the previous release, it comes with three new features:

  • The tab bar shows icons alongside with window captions.
    This feature is adopted from Takashi's fork.

  • The Diff Pane and Location Pane show up on the opposite side when Shift is depressed while invoking the respective menu commands.
    This feature is tracked in Issue #43.

  • A file transform scriptlet was added to Compare As "XML Sorted".
    The scriptlet employs MSXML to apply an XSL transform, and pipes the output through tidy.exe for formatting.
    The included copy of tidy.exe is compiled from a snapshot of Jay Satiro's fork.
    Jay has added long-missed options for less aggressive whitespace munging (https://github.com/w3c/tidy-html5/issues/117).
    This feature is tracked in Issue #31.
